### Start Backend (1 command)
```powershell
cd backend\laravel-5scent; php artisan serve
```

### Start Frontend (1 command)
```powershell
cd frontend\web-5scent; npm run dev
```

### Create Database (2 commands)
```powershell
mysql -u root < setup-db.sql
php artisan migrate
```


```powershell
composer install  # Backend
```
